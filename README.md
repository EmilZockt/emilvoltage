# Coffee Club Design in HTML & CSS and bit of Java
☕ Single-file event website for Coffee Club Bregenz, a Sunday morning day party series at Lake Constance, Austria.
Zero dependencies. No build step, no frameworks, no JavaScript libraries. One HTML file, ~1200 lines. 🧱
Stack: Vanilla HTML/CSS/JS, Google Fonts (Dela Gothic One, DM Mono, Inter), inline SVG venue logos.
Features: ✨ Custom cursor with trailing ring, CSS keyframe animations (sunburst pulse, steam particles, fade-up reveals), IntersectionObserver-driven scroll animations, infinite logo marquee strip, archive grid with embedded video, OpenGraph + Twitter Card meta, fully responsive. 📱
Design: 🇨🇭 Swiss Design influenced. Orange/cream/dark palette, display typography, bold graphic identity.
